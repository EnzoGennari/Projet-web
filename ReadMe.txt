Pour lancer le morpion, lancer simplement le fichier index.html (dans le dossier Morpion)

Pour lancer le chat, lancer le server avec la command "node server.js" (exécutée dans le dossier chat), puis cliquer sur le fichier index.html (dans le dossier chat/www)
